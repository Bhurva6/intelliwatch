# IntelliWatch - AI-Powered Video Intelligence & People Analytics Landing Page

## Overview

IntelliWatch is a modern, professional one-page landing page for an AI-powered video intelligence and people analytics platform. The design is inspired by leading solutions like Spot.ai, Avigilon, and Xovis, combining video security, people flow analytics, and operational intelligence into a unified platform.

## Product Name

**IntelliWatch** - A combination of "Intelligence" and "Watch", conveying smart monitoring and analytics capabilities.

## How to Use

1. **Open the landing page**: Simply open `index.html` in any modern web browser
2. **Navigate**: Click on navigation links to jump to different sections
3. **Request Demo**: Fill out the contact form in the CTA section
4. **Explore**: Scroll through solutions, use cases, and industries

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #4F46E5;
    --secondary-color: #06B6D4;
    /* ... more variables */
}
```

### Content
- Edit section text in `index.html`
- Modify industry cards, use cases, and solutions
- Update company information in the footer

### Form Integration
Connect the form to your backend by modifying the form submission handler in `script.js`:
```javascript
// Replace the console.log with your API call
fetch('your-api-endpoint', {
    method: 'POST',
    body: JSON.stringify(data)
});
```
